# 3. 汪华：垄断的四个发展阶段 {:&.flexbox.vleft}
## 阶段一：找到滩头阵地
互联网的发展变化很快，巨大的变化发生的时候，我们不早也不晚，正好在那个堤坝被抹平的那一瞬间成为第一批，也不是唯一一个，第一批进入新抹平领域的这些人。迅速利用抹平的势能抢滩登陆并占领。

<br/>

分析趋势，变化，政策。例如技术的进步，随智能手机的普及诞生的移动互联网；平台的变化，例如微信这个平台的出现催生的微商；政策的变化，例如看到严惩酒驾的政策，就会意识到代驾是很好的机会。

[slide]
# 3. 汪华：垄断的四个发展阶段 {:&.flexbox.vleft}
## 阶段一：找到滩头阵地
需要注意的点：
* 了解你自己，千万不要做自己不擅长的或者不了解的东西。多思考众多竞争者中凭什么是你能成功？
* 避免多边市场。创业者的资源都是有限的，所以我们选择的市场必须是“万事具备，只欠东风”。
* 复制性壁垒，找到全新的打法实现抄近路，比别人更快的拿下市场。
* 结构化的壁垒，比如早期的小米对传统的联想和华为，电商渠道就是结构化壁垒，他们很难放弃线下渠道。再比如Android系统对于微软的操作系统。

[slide]
# 3. 汪华：垄断的四个发展阶段 {:&.flexbox.vleft}
## 阶段二：拿下滩头阵地
拿下滩头阵地是为了找到自己起始的大市场，找到自己第一个立足点和切入点。从0做到1，目的真的不是只为了做一个1，而是为了从1到N做准备。此阶段最重要，选择切入点一定要选择最有把握赢的点。

<br/>

拿下滩头阵地的真正目的：
* 印证我们的方向，印证我们找到的那个打法
* 累积团队的执行力，找到可以大规模复制的方法论
* 确实我们将来如果要进行大规模复制和增长时候的时候，我们核心KPI是什么。

## 阶段二：拿下滩头阵地
创业者容易犯的错误：
* 跳板思维：我本来是想做B，但是为了把B做起来，我先做A，A做起来再给B倒流。1，创业每个环节有50%做成的可能性，多一个环节成功率呈数量级降低。2，创业者做的很多事情是有时间窗口的，先做A再做B，可能别人就直接把你的B做了。
* 乔布斯陷阱。创业者总觉得自己要做一个特别完美的产品，这样做的问题是公司没法扩张。追求完美需要更高的技术，更复杂的运营，导致自己的扩张成本和扩张速度跟不上别人。最后你只能被哪些做的比较糙但是扩张速度快很多的人干掉（代驾领域的安师傅和e代驾）。

# 4. 傅盛：金山转型的故事 {:&.flexbox.vleft}
1，金山软件的困境：360蒸蒸日上，上市后迅速的成长，每当金山更新了一个版本有了新的功能，360也能很快跟上。

<br/>

2，寻找新的滩头阵地：为了避开360，选择了做移动工具领域、海外市场，切入点选择做清理（Clean master），傅盛分析清理这个产品够简单，容易被大众接受。

[slide]
# 4. 傅盛：金山转型的故事 {:&.flexbox.vleft}
3，拿下滩头阵地，对比现有的产品电池医生（一亿的用户，DNU 20万），最初Clean master是一个四五个人的小团队，DNU从5千很快到了4万，有很大的自增长潜力。傅盛毅然把公司重心转移到Clean master，把所有的推广渠道都找到，让全球用户都知道这个APP。

<br/>

4，复制打法到更多的市场：利用火车头原理横向扩展，14年初推出一年的CM Security下载量达1亿，14年底CM backup有7000万的用户，此外还有CM launcher、CM locker、浏览器等等。猎豹移动于14年中在美国上市。
